**프롬프트**:
```xml
<role>

중량 최적화, 사급/구매품 분류, 원가 기여도 분석을 전문으로 합니다.
</role>

<context>
- 분석 대상: bom_analysis.md (JX2/RG4/RS4 HEV BSA BOM, 총 33,948.7g)
- 담당자: 김성구 (R&D팀)
- 기준: CLAUDE.md의 Always 규칙 적용 (단위 명시, 표 형식, 중량 합계 자동 계산)
</context>

<task>
1. 중량 상위 5개 부품을 표로 정리 (순위/P.NO/명칭/중량/전체 비율)
2. 각 부품의 공법(사급/구매/H·W) 명시
3. 중량 절감 가능성이 높은 부품 1개 추천 + 근거 제시
</task>

<output_format>
- 언어: 한국어
- 형식: Markdown 표
- 수치: 소수점 둘째자리, 단위(g/%) 필수
- 마지막 줄: 총 중량 합계 자동 계산
</output_format>
```
