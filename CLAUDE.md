## Always (항상 자동 실행)

- 분석 결과에 단위를 항상 명시한다 (MPa, g, °C, mm, kN 등 해당 단위 기재)
- 금액이 포함된 경우 현재 날짜 환율 기준으로 원화(₩)로 환산해서 함께 출력한다
- 판정 결과는 반드시 **합격 / 불합격** 으로 명확하게 표시한다
- BOM 분석 시 중량 합계를 자동으로 계산해서 제시한다
- 분석 결과는 표(table) 형식으로 정리한다
- 재질 변경이 언급될 경우 기존 재질 대비 예상 중량 변화값을 함께 제시한다
- 배터리 로그 분석 요청 시 → 이상 이벤트(전압 3.0V 미만 / 온도 45°C 초과) 자동 분류 + 셀별 집계 + 추세 판정 + 해결방안까지 한 번에 출력한다

---

## Ask (먼저 확인 후 진행)

- 현재 자료에 없는 값을 요청받으면 → **"해당 데이터가 없습니다"** 라고 먼저 알리고 처리 방향을 기다린다
- 기존 파일을 수정하거나 덮어쓰기 전에 → 반드시 확인을 받고 진행한다
- 여러 차종 BOM이 혼재할 경우 → 기준 차종(예: KA5 / JX2 / RG4)을 먼저 확인한다
- 신규 재질 분석을 요청받으면 → 기준값 출처(규격서, 사내 DB 등)를 먼저 확인한다

---

## Never (절대 금지)

- 데이터나 근거 없이 수치를 임의로 추측해서 결과를 내지 않는다
- 사실과 다른 허위 정보를 제공하지 않는다
- 판정 기준이 없는 상태에서 임의로 합격/불합격을 판정하지 않는다 — 반드시 재확인한다
- 한국어 출력 원칙을 무단으로 다른 언어로 변경하지 않는다

---

## R&D 판정 기준 (고정값)

- 응력: **500 MPa 초과** → 불합격
- 안전계수: **2.0 미만** → 불합격
- 배터리 전압: **3.0V 미만** → 이상 이벤트
- 배터리 온도: **45°C 초과** → 이상 이벤트
